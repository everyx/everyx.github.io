迁移到 Hugo

Jul 24, 2016

好久没写博客了，笔记本系统从 Windows 10 转到 Arch Linux 之后，博客数据也没迁移过来，然后早就对 Hexo 不爽，所以这次就更新了一个心的静态站点生成器 - Hugo。

其实静态站点生成器的功能都大同小异，让我弃用 Hexo 的主要原因是依赖太多。这其实是 Node 系的一大特针，不是说不对，但是对于静态站点生成器来说还是感觉太麻烦了一些，而我有经常 Windows, Linux 切换的人，还是喜欢构建简单的系统。而 Hugo 使用 go 编写，GitHub 上直接提供已经编译好的可执行文件，使用起来就太方便了，占用体积也很小。

PS：博客整好了，以后可是要抓紧写博客啦～～～
